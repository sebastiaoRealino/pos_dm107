### Run!


Instalação do Composer.

```sh
curl -s https://getcomposer.org/installer | php
```

Instalação do Slim.

```sh
$ php composer.phar require slim/slim
```

Dependencias do NotORM

```sh
$ php composer.phar require vrana/notorm:dev-master
```
